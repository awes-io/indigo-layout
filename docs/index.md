# Indigo Layout

<p align="center">
  <img src="https://static.awes.io/docs/awes-io.png" alt="Awes.IO" />
</p>

## Install

```bash
composer require awes-io/indogo-layout
```

## Example
```blade
@extends('indigo-layout::auth')
 
@section('title')
    <h1>Awesome App!</h1>
@endsection
 
@section('content')
    ...
@endsection
 
@section('footer')
    ...
@endsection
```
